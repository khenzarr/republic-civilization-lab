## Creating Demo GIF

To create the demo GIF:

1. **Record Screen**
   - Use tools like: ScreenToGif, LICEcap, or Kap
   - Record at 1920x1080 or 1280x720
   - Keep duration under 10 seconds
   - Show key features: input, generation, result

2. **Optimize**
   - Reduce file size to < 5MB
   - Use tools like: ezgif.com or gifsicle
   - Maintain quality while reducing size

3. **Content to Show**
   - Homepage with animated background
   - Typing a theme
   - Clicking generate button
   - Loading state
   - Result display with civilization card
   - Copy/download functionality

## Screenshot Guidelines

### Homepage Screenshot
- Show full page with header
- Include animated background
- Display input form
- Show footer with branding

### Result Screenshot
- Show generated civilization card
- Include language table
- Display mythology section
- Show export buttons

## Image Specifications

- **Format**: PNG for screenshots, GIF for animations
- **Resolution**: 1920x1080 (desktop), 375x812 (mobile)
- **Compression**: Optimize for web
- **Naming**: Use kebab-case (e.g., `demo-placeholder.gif`)
